VARIABLES IN PYTHON

Variables are reserved memory locations to store values. By assigning different data types to the variables, we can store integers, decimals or characters in these variables. However in python we don't need to declare the data type, the declaration happens automatically when we assign a value to a variable.

The operand to the left of the ' = ' is the name of the variable and the operand to the right of the ' = ' is the value stored in the variable.
